Convertigo Studio
=================

Convertigo Studio installer is installing the Convertigo Studio software
product.

Convertigo Studio is based on Affero GPL V3 License.

Convertigo Studio is for "Convertigo Project" developments purposes. Convertigo
Projects are to be executed on "Convertigo Server".

Convertigo Studio includes modules such as Convertigo Mobilizer, Convertigo
Widgetizer, Convertigo Sequencer and SQL, REST/SOAP connectors.

Convertigo Enterprise Edition components delivered and bundled with Convertigo
Studio are free of use for developments purpose only. They include HTML and
Mainframe connector.


Convertigo Server
=================

Convertigo Server installer is installing the Convertigo Server software
product.

Convertigo Server is based on Affero GPL V3 License.

Convertigo Server is for production purposes and executes Convertigo projects
developed with Convertigo Studio.

Convertigo Server includes Convertigo Mobilizer, Convertigo Widgetizer,
Convertigo Sequencer and SQL, REST/SOAP connectors.

Convertigo Enterprise Edition components delivered and bundled with Convertigo
Server are pre activated for a 30 days trial period and requires keys to extend
the activation period.

Non-key activated Convertigo Enterprise Edition Components delivered and bundled
with Convertigo Studio and Convertigo Server are inactive and can be copied and
distributed only bundled with Convertigo products.
